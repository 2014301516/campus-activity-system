## 一、课题清单表
| 序号 | 课题名称 | 推荐前端框架 | 后端要求 | 主要功能 |
| ---- | ---- | ---- | ---- | ---- |
| 1 | 校园失物招领系统 | Vue / React | 不限 | 注册登录、发布、列表、搜索、认领 |
| 2 | 图书借阅管理系统 | Vue / React | 不限 | 注册登录、图书列表、借阅、归还、统计 |
| 3 | 简易论坛 | Vue / React | 不限 | 注册登录、发帖、评论、分类、搜索 |
| 4 | 餐厅点餐系统 | Vue / React | 不限 | 注册登录、点餐、订单管理、付款等 |
| 5 | 校园活动管理系统 | Vue 3 | Spring Boot + MySQL | 登录、活动发布、审核、报名、签到、签退、评价、统计 |

## 二、通用开发要求
1. 系统页面数量：总计 **5—8个页面**。
2. 数据库设计：至少设计 **4张核心数据表**。
3. 用户角色：系统需包含管理员、普通用户等多种角色，并区分权限。
4. 基础功能：完整实现数据**增、删、改、查**操作。
5. 演示安排：倒数第二节课进行项目演示，演示包含答辩环节。
6. 文档要求：按规范填写各类配套表格，附系统界面截图，并说明使用的AI工具及使用情况。

## 配套表格模板

### 本次选题

- 项目名称：校园活动管理系统
- 前端技术：Vue 3 + Vite + Element Plus + Pinia + Vue Router + Axios + ECharts
- 后端技术：Spring Boot + MyBatis-Plus + Spring Security + JWT
- 数据库：MySQL 8.0
- 开发模式：前后端分离
- 用户角色：学生、组织者、管理员

### 4. 接口设计表
| 接口名称 | 请求方式 | 地址 | 参数 | 返回结果 | 说明 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| 注册 | POST | /api/user/register | 用户名、密码、姓名、角色 | 注册结果 | 用户注册 |
| 登录 | POST | /api/user/login | username, password | token, userInfo | 用户登录 |
| 获取活动列表 | GET | /api/activity/list | keyword、categoryId、status | 活动列表 | 活动查询 |
| 获取活动详情 | GET | /api/activity/{id} | 活动 ID | 活动详情 | 查看活动信息 |
| 创建活动 | POST | /api/activity | 标题、描述、时间、地点、人数等 | 创建结果 | 组织者发布活动 |
| 更新活动 | PUT | /api/activity/{id} | 活动 ID、修改字段 | 更新结果 | 编辑活动 |
| 删除活动 | DELETE | /api/activity/{id} | 活动 ID | 删除结果 | 删除活动 |
| 报名活动 | POST | /api/activity/{activityId}/register | 活动 ID | 报名结果 | 学生报名 |
| 取消报名 | DELETE | /api/activity/{activityId}/register | 活动 ID | 取消结果 | 学生取消报名 |
| 获取我的报名 | GET | /api/my-registrations | 无 | 报名记录列表 | 学生查看报名记录 |
| 签到 | POST | /api/sign-in | activityId | 签到结果 | 学生签到 |
| 签退 | PUT | /api/sign-in/{id} | 签到记录 ID | 签退结果 | 学生签退 |
| 提交评价 | POST | /api/review | activityId、rating、comment | 提交结果 | 活动评价 |
| 获取公告列表 | GET | /api/notice/list | 无 | 公告列表 | 首页公告展示 |
| 获取后台统计 | GET | /api/dashboard/stats | 无 | 统计结果 | 后台概览 |
| 获取用户列表 | GET | /api/admin/users | page、size | 用户列表 | 管理员用户管理 |
| 审核活动 | PUT | /api/admin/activity/{id}/audit | 活动 ID、审核状态 | 审核结果 | 管理员审核活动 |
| 新增分类 | POST | /api/admin/category | 名称、描述 | 新增结果 | 管理员新增分类 |
| 发布公告 | POST | /api/admin/notice | 标题、内容 | 发布结果 | 管理员发布公告 |
